# Sunsetr

My Sunsetr settings

Repo:  
https://github.com/psi4j/sunsetr

Docs:  
https://github.com/psi4j/sunsetr/blob/main/docs/src/SUMMARY.md

Copy the contents of this repository to `~/.config/hypr`.
The following files can be ignored:

- `README.md`
- `.git`
- `.gitignore`

## Setup

1. Install the package:

```sh
yay -S sunsetr
```

2. Set up location:

```sh
sunsetr geo # Select current location from list, or the nearest city
```
